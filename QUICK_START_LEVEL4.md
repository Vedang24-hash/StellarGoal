## 💡 Pro Tips

### For User Onboarding:
- Offer to help with wallet setup (screen share)
- Create a simple "Quick Start" guide for users
- Fund their wallets yourself if needed (testnet XLM is free)
- Make it fun - gamify with "first user" recognition

### For Feedback:
- Keep survey short (5-10 questions max)
- Mix multiple choice and open-ended
- Ask specific questions about features
- Incentivize with acknowledgment in README

### For Demo Video:
- Use high quality microphone
- Speak slowly and clearly
- Show cursor movements
- Add text overlays in editing
- Keep it under 3 minutes
- Practice once before recording

**"Can't find 10 users"**

- Use crypto developer communities

- Record at 1080p resolution

- Re-record if needed (doesn't take long)
